## 👥 팀 구성

| 역할 | 이름 |
| --- | --- |
| **PM** | 이재민 |
| **PD** | 주현지 |
| **Frontend** | 김규태, 김민지, 엄현용 |
| **Backend** | 노영서, 정다운, 장효원 |

---

## 🛠 기술 스택

### Frontend

- React.js
- TypeScript
- Tailwind CSS

### Backend

- Spring Boot
- JPA
- MySQL
- Redis
- Docker

### 인증 & 배포

- OAuth2 (Google 로그인)
- AWS, Vercel
